autoSlider.js

a simple auto image slider.

This is a simple auto image slider jQuery plugin.
It supports two simple animation styles as of now which are Ping Pong & Scroll From First.

Usage:

Markup:

<div id="carousel">
  <div id="slides">
    <div>1</div>
    <div>2</div>
    <div>3</div>
    <div>4</div>
  </div>
</div>

Javascript:

$('#carousel').autoSlider({
  slidesContainerId: 'slides',
  animation: 'pingPong',
  interval: 2000
});

CSS:

#carousel > #slides > div {
  height: pixels_here,
  width: pixels_here
}
